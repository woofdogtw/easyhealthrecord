EasyHealthRecord Droid can help you to record common meter values.


EasyHealthRecord Droid can help you to record common values of body fat, blood pressure, blood glucose meters, and backup records to your network space.

- FTP, Microsoft OneDrive online backup mechanism.
- Using file copy to backup and restore files.
- We also provides dedicated Windows desktop and Windows Store App programs (visit website).

This program does not support CSV exporting. You can
(1) Visit the website (https://easyhealthrecord.firebaseapp.com) and download Windows desktop version or
(2) Visit Windows Store or Windows Phone Store and download EasyHealthRecord RT
to get full functions.
